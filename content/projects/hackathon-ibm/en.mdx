## Context

Project built during an IBM hackathon centered around Business Intelligence. I led a team of 4
developers with the mission of designing an intelligent question-answering system powered by an
existing knowledge base.

## Problem Statement

Enable students at Pôle Léonard de Vinci to query a knowledge base (help center) in natural
language and receive synthesized, sourced answers without having to manually browse
through articles.

## Solution

Implementation of a **RAG (Retrieval-Augmented Generation)** system combining a relational
database, a vector search engine and a LLM:

- **Ingestion**: help center content is stored in MariaDB and indexed as vectors in
  Elasticsearch via the OpenAI embeddings API (`text-embedding-3-small`).
- **Semantic search**: for each user query, cosine similarity is computed to retrieve the
  most relevant passages.
- **Synthesis**: GPT-4o-mini generates a coherent response from the retrieved documents,
  with source citations.
- **Feedback loop**: user feedback is recorded to monitor response quality.
